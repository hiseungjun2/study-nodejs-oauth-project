# 초격차 패키지 : 한 번에 끝내는 Node.js 웹 프로그래밍
# OAuth 인증 실습

### 1. Package install
 ```
 npm install
 ```

### 2. ngrok 설치 후 실행
 ```
 ngrok http 5000
 ```

### 3. .env 생성 후 파라미터 세팅
 ```
 FB_APP_ID=
 FB_CLIENT_SECRET=
 SERVER_SECRET=
 NAVER_CLIENT_ID=
 NAVER_CLIENT_SECRET=
 KAKAO_JAVASCRIPT_KEY=
 KAKAO_REST_KEY=

 HOST=
 PORT=
 NODE_ENV=

 MONGO_USER=
 MONGO_PASSWORD=KSJ=
 MONGO_CLUSTER=
 MONGO_DBNAME=
 ```
